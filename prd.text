1. Product Overview
Goal

Create a modern, fast, offline-first Indian recipe app containing 10,000–50,000 recipes covering all Indian cuisines.

The app should help users:

Discover recipes

Cook with step-by-step instructions

Plan meals

Generate grocery lists

Ask AI cooking questions

The app must be:

Fast

Clean UI

Lightweight

Offline capable

Free

No ads

No subscription

2. Target Users
Beginner cooks

Need simple recipes with steps.

Home cooks

Need meal planning and grocery lists.

Students / busy users

Need quick recipes and filters.

Health conscious users

Need nutrition data and diet filters.

7. Database Design

Since you want JSON recipes, each recipe should follow strict schema.

Recipe JSON structure
{
  "id": "paneer_butter_masala",
  "name": "Paneer Butter Masala",
  "cuisine": "Punjabi",
  "diet": "Vegetarian",
  "difficulty": "Medium",
  "cook_time": 35,
  "servings": 4,
  "calories": 320,
  "rating": 4.8,
  "reviews": 1200,
  "image": "paneer_butter_masala.jpg",

  "ingredients": [
    {
      "name": "Paneer",
      "quantity": 200,
      "unit": "g"
    },
    {
      "name": "Tomato",
      "quantity": 2,
      "unit": "pcs"
    }
  ],

  "preparation": [
    {
      "ingredient": "Onion",
      "cut": "finely chopped"
    },
    {
      "ingredient": "Paneer",
      "cut": "cube"
    }
  ],

  "nutrition": {
    "protein": 12,
    "carbs": 18,
    "fat": 20,
    "fiber": 3,
    "sugar": 4
  },

  "equipment": [
    "Pan",
    "Blender",
    "Knife"
  ],

  "steps": [
    {
      "step": 1,
      "instruction": "Heat butter in a pan",
      "time": 3
    },
    {
      "step": 2,
      "instruction": "Add chopped onion and sauté",
      "time": 5
    }
  ],

  "tips": [
    "Use fresh cream for better taste",
    "Roast paneer for smoky flavor"
  ],

  "tags": [
    "quick",
    "indian",
    "vegetarian"
  ]
}
